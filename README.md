# Student Portal

A simple student portal web project built using HTML and CSS.  
It displays available courses and allows users to click on a course to view detailed information.

## Features
- Course dashboard
- Clickable course cards
- Separate course detail pages
- Simple and clean UI

## Technologies
- HTML
- CSS
- Git & GitHub

## How to Run
Open `index.html` in a web browser.
